# Rational
Rational implementation in C++ to perform rational arithmetics

Platforms:
----------
    Linux

Run & Compile:
-------------
    to compile:
        $ make
    to run:
        $ ./main

Contructors:
------------
    default contructor:
    Rational <int> a; // a = 0/1

    contructor with denominator and numerator:
    Rational <int> x(1, 2);

Getters:
--------------------
    den()
    num()

Operators:
----------
    +
    -
    /
    *

Comparison:
-----------
    >
    <
    >=
    <=
    ==
    !=

IO Operator:
------------
    <<
    >>
